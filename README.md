# escape-room-programacao
Escape Room - Fuja da sala

-História do Jogo
Voce ficou trancado em uma sala de servidores do CEUB e agora tem que descobrir uma senha de 1 ate 100 para escapar.

-Como Jogar
Em seu terminal execute o codigo e tente adivinhar a senha correta com ajuda das dicas que seram dadas a cada tentativa.
